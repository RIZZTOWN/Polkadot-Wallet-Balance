# Blockchain Wallet Balance

Blockchain Wallet Balance gets the balance of a wallet address on a blockchain

## Installation

Use the package manager npm to install Blockchain Wallet Balance from root directory

```bash
npm install
```

## Usage

```bash
# To build & start
npm run start:build

# To start
npm run start

# To build
npm run build

# To run dev
npm run dev

# To test
npm run test
```

- Polkadot
